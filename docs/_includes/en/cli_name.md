??? info "`mybible-cli` in CLI Examples"
    In most command-line usage examples in this documentation, we use `mybible-cli` for brevity and consistency.

    **Your actual command may differ depending on your installation:**

    - **JAR file**: `java -jar mybible-cli.jar`
    - **Windows (Console version)**: `mybible-cli-console.exe` or helper scripts (`mybible-cli.ps1`, `mybible-cli-unicode.bat`, etc.)
    - **Linux/macOS**: `/path/to/mybible-cli` (full path to the binary)

    Regardless of how you invoke the application, all options and subcommands work identically across platforms.
