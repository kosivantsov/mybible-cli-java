# GUI Keyboard Shortcuts

The `mybible-cli` graphical user interface (GUI) provides several keyboard shortcuts to speed up your workflow and provide quick access to common actions. These shortcuts help you navigate the application and perform tasks without using the mouse.

| Windows / Linux  | macOS           | Action                                                |
| :--------------- | :-------------- | :---------------------------------------------------- |
| ++esc++          | ++esc++         | Quit the application                                  |
| ++ctrl+l++       | ++cmd+l++       | Focus the "Bible reference" input field               |
| ++enter++        | ++enter++       | Fetch verses for the reference in the input field     |
| ++ctrl+m++       | ++cmd+m++       | Focus the "Module" dropdown list                      |
| ++ctrl+j++       | ++cmd+j++       | Toggle raw JSON output                                |
| ++ctrl+shift+c++ | ++cmd+shift+c++ | Copy displayed text (with rich text formatting)       |
|                  | ++cmd+comma++   | Open the Configuration dialog                         |
