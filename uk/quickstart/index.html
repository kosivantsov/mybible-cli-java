    Quick Start Guide¶

    Get up and running with mybible-cli in minutes.

    Prerequisites¶

    To start using mybible-cli, you need to make sure that:

    1. It's present on your computer – see Installation Guide.
    2. You have at least one unpacked MyBible module – see MyBible Modules.

    First-time Setup¶

    The only absolutely necessary thing that needs to be done prior to the first use is telling the application where your MyBible modules are stored.

    If you have one folder with modules, this needs to be done only once.

    If you have multiple folders with modules and want to switch between them, setting the module path needs to be done once before each switch.

    The application uses the same configuration file regardless of the mode it runs in (CLI or GUI) or how it was launched (from terminal, in a script, as a JAR file, as a compiled OS-specific application, etc.).

    mybible-cli у прикладах використання у командному рядку

    У цій документації у прикладах з використання програми у командному рядку назва команди подається здебільшого як mybible-cli для стислості та послідовності.

    У дійсності команда, яку ви маєте запускати, може бути іншою залежно від інсталяції:

    • Файл JAR: java -jar mybible-cli.jar
    • Windows (консольна версія): mybible-cli-console.exe або допоміжні скрипти (mybible-cli.ps1, mybible-cli-unicode.bat тощо).
    • Linux/macOS: /шлях/до/mybible-cli (повний шлях до бінарного файлу) Як би ви не викликали програму, всі опції та підкоманди працюють однаково на всіх платформах.

    Встановити шлях до модулів¶

    mybible-cli list --path "/path/to/your/MyBible/modules"

    Example paths:

    mybible-cli list --path "C:\Users\YourName\Documents\MyBible\modules"

    mybible-cli list --path "$HOME/Documents/MyBible/modules"

    mybible-cli list --path "$HOME/MyBible/modules"

    List Available Modules¶

    mybible-cli list

    You should see your modules sorted by language:

    1. Launch the GUI by opening the application from the Start menu, double-clicking the JAR file, or from the terminal: 
      mybible-cli

    2. Click "Set Module Path".

    3. If you selected the correct folder, you will be able to see the list of modules sorted by language.

    Setting the module path needs to be done only once. The path will be saved for future use.

    Switch module folder

    If you need to switch to a different module folder, simply repeat the above and specify the folder you want to use.

    Basic Usage¶

    Fetch a Single Verse¶

    mybible-cli get -m KJV+ -r "John 3:16"

    Output: 

    John 3:16 For God so loved the world, that he gave his only begotten Son, that whosoever believeth in him should not perish, but have everlasting life.

    Fetch Multiple Verses¶

    mybible-cli get -m ESV -r "Romans 8:28-30"

    Fetch Multiple Passages¶

    mybible-cli get -m NET -r "John 3:16; Romans 8:28; Ephesians 2:8-10"

    Use Last Module¶

    # After first use, you can omit -m
mybible-cli get -r "Psalm 23"

    Launch the GUI by opening the application from the Start menu, double-clicking the JAR file, or from the terminal:

    mybible-cli

# Or with a pre-loaded passage:
mybible-cli gui -m KJV+ -r "John 3:16"

    In the GUI:

    1. Select a module from the dropdown.
    2. Enter a reference (e.g., "John 3:16", "Romans 8:28-30", or "John 3:16; Romans 8:28; Ephesians 2:8-10").
      This can be one verse, several verses, several chapters, or several passages.

    3. Press Enter or click "Show".

    Common Tasks¶

    Read a Chapter¶

    mybible-cli get -m ESV -r "Psalm 23"
    1. Press Ctrl+L / Cmd+L or click in the "Bible Reference" field.
    2. Enter the reference, and change the module if needed.
    3. Press Enter or click "Show".

    Read Multiple Chapters¶

    mybible-cli get -m KJV -r "Matthew 5-7"
    1. Press Ctrl+L / Cmd+L or click in the "Bible Reference" field
    2. Enter the reference, and change the module if needed
    3. Press Enter or click "Show".

    Copy Output to Clipboard¶

    mybible-cli get -m NET -r "Romans 8:28" | clip

    mybible-cli get -m NET -r "Romans 8:28" | pbcopy

    mybible-cli get -m NET -r "Romans 8:28" | xclip -selection clipboard
    1. Type the required reference and select the module.
    2. Press Enter or click "Show".

    3. Press Ctrl+Shift+C / Cmd+Shift+C or click "Copy Text".

    Save to File¶

    mybible-cli get -m KJV -r "Psalm 119" > psalm119.txt

    1. Type the required reference and select the module.
    2. Press Enter or click "Show".
    3. Press Ctrl+Shift+C / Cmd+Shift+C or click "Copy Text".
    4. Launch a text editor or a word processor, and create a new document.
    5. Paste. If the application supports extended formatting, some formatting will kept.
    6. Save the newly created document.

    Get JSON Output¶

    mybible-cli get -m NET -r "John 3:16" --json

    Output: json [ { "bookNumber": 500, "defaultFullBookName": "John", "defaultShortBookName": "John", "moduleFullBookName": "John", "moduleShortBookName": "Jn", "allBookNames": ["John", "Jhn", "Jn"], "chapter": 3, "verse": 16, "rawVerseText": "<pb/> <J>For this is the way <f>[36]</f> God loved the world: He gave his one and only <f>[37]</f> Son, so that everyone who believes in him will not perish <f>[38]</f> but have eternal life.</J> <f>[39]</f>", "moduleName": "NET" } ]

    1. Type the required reference and select the module.
    2. Press Enter or click "Show".

    3. Press Ctrl+J / Cmd+J or mark the "Show Raw JSON" checkbox.
